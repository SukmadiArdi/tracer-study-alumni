# Alumni Tracer Study System

Web-based tracer study system used to track alumni career outcomes.

## Features

- Admin Login
- Alumni Management
- Alumni Tracking (OSINT Simulation)
- Manual Verification
- Dashboard Statistics
- Cloud Database

## Technology

- HTML
- TailwindCSS
- JavaScript
- Firebase Firestore
- Chart.js
