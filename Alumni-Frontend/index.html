Lengkapi data sosial media & pekerjaan alumni secara semi-otomatis

Cara Penggunaan Fitur Enrichment

  1. Pergi ke menu Data Alumni
  2. Klik tombol Enrichment pada baris alumni
  3. Gunakan tombol pencarian otomatis untuk buka LinkedIn, IG, FB, TikTok
  4. Salin URL profil yang ditemukan ke dalam form
  5. Isi data pekerjaan — status PNS/Swasta/Wirausaha akan otomatis terdeteksi
  6. Klik Simpan Data untuk menyimpan ke database
